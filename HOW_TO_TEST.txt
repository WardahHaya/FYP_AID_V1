NEXT STEPS FOR ENABLING GROQ API 🔑
================================================================================

To get REAL LLM-generated reports (not just mock text):

1. Get Groq API Key:
   Visit: https://console.groq.com/keys
   Sign up if needed, then copy your API key

2. Add to .env file:
   echo "GROQ_API_KEY=your_actual_key_here" >> .env

3. Restart the server:
   Kill current: Ctrl+C (in the terminal running uvicorn)
   Restart: python3 -m uvicorn server.app:app --reload --host 0.0.0.0 --port 8000

4. Test:
   python3 test_report_generation.py
   (Will now show real LLM output instead of prompt structure)

================================================================================
TROUBLESHOOTING 🐛
================================================================================

Q: PDFs not generating?
A: Check ReportLab: pip install -r requirements.txt

Q: Urdu text looks wrong?
A: Ensure RTL libs: pip install arabic-reshaper python-bidi

Q: LLM report is placeholder text?
A: Set GROQ_API_KEY in .env, then restart server

Q: "ModuleNotFoundError"?
A: Run: python3 -m py_compile src/*.py (to identify issue)

Q: Server errors on startup?
A: Check: python3 src/llm_report_groq.py (direct import test)
